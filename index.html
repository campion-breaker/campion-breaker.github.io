Overview

What Is It?

Campion is a free, open-source tool to help protect your site or service from dependency failure. It does this by implementing circuit breaking, a concept that utilizes an automatic fail fast mechanism when a dependency is down.

 

Easy Setup

Deploy your own circuit breaker middleware running in seconds using campion setup or campionaws setup.
